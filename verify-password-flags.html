Step 1: Check Database Flags

After creating a test user, you can verify the flags are set correctly by checking the database:

-- Check profiles table for temp_password flag SELECT email, temp_password FROM profiles WHERE email = 'testuser@vitaluxe.test'; -- Check user_password_status table SELECT user_id, must_change_password, temporary_password_sent, first_login_completed FROM user_password_status WHERE user_id = (SELECT id FROM profiles WHERE email = 'testuser@vitaluxe.test');

Step 4: Test the Complete Flow

Follow these steps to test the complete flow:

  1. Create test user using Test Password Manager
  2. Check database flags - should be set to true
  3. Try to login with temp password - should be blocked
  4. Change password - should work
  5. Check database flags again - should be cleared
  6. Login with new password - should work

Step 7: Troubleshooting

If something doesn't work as expected:

  • Check browser console for any JavaScript errors
  • Check network tab for failed API calls
  • Verify database flags using the SQL queries above
  • Check server logs for any backend errors
