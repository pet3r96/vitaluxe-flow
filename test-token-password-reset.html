This test verifies the complete token-based password reset flow:

  1. Send Welcome Email - Trigger welcome email with token
  2. Validate Token - Check if token is valid and get user info
  3. Test Password Change - Simulate password change via token
  4. Test Auto-Login - Verify automatic login after password change

Manual Testing Steps

Complete Manual Test Flow

  1. Create Test User: Use admin panel to create a new user account
  2. Send Welcome Email: Use the "Resend Welcome Email" button in account management
  3. Check Email: Verify email contains only the "Set Your Password" button (no temporary password shown)
  4. Click Button: Click the button in the email
  5. Set Password: Enter a new password that meets requirements
  6. Verify Auto-Login: User should be automatically logged in and redirected to dashboard
  7. Test Login: Log out and log back in with the new password to verify it works
